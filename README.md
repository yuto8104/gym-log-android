# GymLogApp

Kotlin / Jetpack Compose で制作した、Android向けのトレーニング記録アプリです。

## 概要

種目を選択し、重量・回数を入力してトレーニング内容を記録できます。  
また、選択した種目ごとにおすすめのアフターケアや食事アドバイスを表示します。

## 主な機能

- 部位カテゴリ別の種目選択
- 重量・回数の入力
- トレーニング記録の追加
- 記録一覧表示
- 記録削除
- SharedPreferences による記録保存
- 種目ごとのアフターケア表示
- 種目ごとの食事アドバイス表示
- 入力画面 / 記録一覧画面のタブ切り替え

## 使用技術

- Kotlin
- Jetpack Compose
- Android Studio
- SharedPreferences
- JSON

## 工夫した点

- 手入力だけでなく、種目リストから選択できるようにした
- 記録画面と一覧画面をタブで分離し、見やすさを改善した
- アプリを閉じても記録が残るように保存機能を実装した
- 単なる記録アプリではなく、アフターケアや食事アドバイスも表示する補助アプリにした
